Descrição do Projeto
Este projeto consiste em um programa interativo em Python que fornece informações educativas e entretenimento sobre a Fórmula E. Ele inclui funcionalidades como um quiz interativo, estatísticas de pilotos e equipes, e um simulador de corridas. O objetivo é proporcionar uma experiência envolvente para os fãs de automobilismo e promover o conhecimento sobre a Fórmula E.

Funcionalidades
Iniciar Quiz: Teste seus conhecimentos sobre a Fórmula E com perguntas desafiadoras.
Ver Pontuação do Quiz: Confira a pontuação obtida no quiz.
Estatísticas de Pilotos e Equipes: Visualize estatísticas detalhadas sobre os pilotos e equipes.
Simulador de Corridas: Simule os resultados de uma corrida da Fórmula E.
Sair: Encerre o programa.

Detalhes das Funcionalidades
1. Iniciar Quiz
Descrição: O quiz apresenta perguntas sobre a Fórmula E. O usuário deve escolher a opção correta entre quatro alternativas.
Como usar: Selecione a opção "1" no menu principal para iniciar o quiz. Responda as perguntas inserindo "A", "B", "C", ou "D".
2. Ver Pontuação do Quiz
Descrição: Exibe a pontuação atual do usuário no quiz.
Como usar: Selecione a opção "2" no menu principal para ver a pontuação.
3. Estatísticas de Pilotos e Equipes
Descrição: Fornece informações detalhadas sobre os pilotos e equipes, incluindo vitórias, pódios e pontos acumulados.
Como usar: Selecione a opção "3" no menu principal para visualizar as estatísticas.
4. Simulador de Corridas
Descrição: Simula os resultados de uma corrida da Fórmula E, gerando um resultado aleatório entre várias possibilidades.
Como usar: Selecione a opção "4" no menu principal para simular uma corrida.
5. Sair
Descrição: Encerra o programa.
Como usar: Selecione a opção "5" no menu principal para sair do programa.
